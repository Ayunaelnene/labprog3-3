102 Etiquetas HTML (estándar HTML5 – 2025/2026)

# Etiqueta Descripción Ejemplo de uso Notas / Elemento adicional
1 <!DOCTYPE html> Declara que el documento es HTML5 (obligatorio al inicio) <!DOCTYPE html> Sin cierre. Activa modo estándares.
2 <!-- --> Comentario (no visible en la página) <!-- Nota para desarrolladores --> Puede abarcar varias líneas.
3 <a> Hipervínculo / enlace <a href="https://ejemplo.com" target="_blank">Visitar</a> atributos: href, target, rel="noopener noreferrer", download
4 <abbr> Abreviatura o acrónimo <abbr title="HyperText Markup Language">HTML</abbr> title muestra tooltip (accesibilidad)
5 <address> Información de contacto del autor/propietario <address>info@empresa.com</address> Semántico – suele ir en footer
6 <area> Área clickable en mapa de imagen <area shape="rect" coords="0,0,100,100" href="zona.html" alt="Zona"> Dentro de <map>
7 <article> Contenido independiente (post, noticia, artículo) <article><h2>Título</h2><p>Texto...</p></article> HTML5 semántico
8 <aside> Contenido tangencial (sidebar, nota al margen) <aside><h3>Dato curioso</h3>...</aside> HTML5 semántico
9 <audio> Reproductor de audio <audio controls><source src="cancion.mp3" type="audio/mpeg"></audio> controls, autoplay, loop, muted
10 <b> Texto en negrita (sin énfasis semántico) <b>Importante</b> Preferir <strong> para importancia
11 <base> Especifica la URL base para todos los enlaces relativos de la página <base href="https://ejemplo.com/"> Debe ir dentro de <head>. Solo se permite uno por documento.
12 <bdi> Aísla una porción de texto para que no afecte la direccionalidad del texto circundante Usuario <bdi>احمد</bdi> escribió:</code> Muy útil en textos con mezcla de LTR y RTL (ej. árabe, hebreo)
13 <bdo> Sobreescribe la dirección del texto (bidirectional override) <bdo dir="rtl">Este texto se lee de derecha a izquierda</bdo> dir puede ser "ltr" o "rtl"
14 <blockquote> Define una cita larga (generalmente con indentación) <blockquote cite="https://...">Texto de la cita...</blockquote> El atributo cite es opcional pero recomendado
15 <body> Contiene todo el contenido visible de la página <body> ... todo el contenido visible ... </body> Solo puede haber uno por documento HTML
16 <br> Inserta un salto de línea Línea uno<br>Línea dos Elemento vacío (sin cierre). No usar para espaciado (mejor CSS)
17 <button> Define un botón clickable <button type="submit">Enviar formulario</button> type puede ser: button, submit, reset
18 <canvas> Área para dibujar gráficos mediante JavaScript <canvas id="lienzo" width="400" height="300"></canvas> Proporciona fallback dentro del tag para navegadores sin soporte
19 <caption> Título o descripción de una tabla <table><caption>Ventas 2025</caption>...</table> Mejora accesibilidad. Debe ser el primer hijo de <table>
20 <cite> Indica el título de una obra creativa (libro, película, etc.) Según <cite>El Principito</cite>, ... Generalmente se muestra en cursiva
21 <code> Define un fragmento de código de computadora <code>console.log("Hola mundo")</code> Suele combinarse con <pre> para bloques
22 <col> Especifica propiedades de una columna en una tabla <col span="2" style="background:#f0f0f0"> Va dentro de <colgroup>
23 <colgroup> Agrupa columnas de una tabla para darles estilos comunes <colgroup><col><col span="2"></colgroup> span define cuántas columnas afecta
24 <data> Vincula un contenido legible con un valor máquina-legible <data value="978-3-16-148410-0">ISBN 978-3-16-148410-0</data> Útil para scripts y microdatos
25 <datalist> Lista de opciones predefinidas para un <input> <input list="browsers"><datalist id="browsers"><option>Chrome</option>...</datalist> Funciona como sugerencias/autocompletado
26 <dd> Describe un término en una lista de descripción <dd>Descripción del término</dd> Se usa junto con <dt> dentro de <dl>
27 <del> Texto que ha sido eliminado de un documento <del datetime="2025-12-01">Precio viejo</del> Suele mostrarse tachado. Atributos cite y datetime opcionales
28 <details> Crea un widget de divulgación (expandir/colapsar) <details><summary>¿Pregunta?</summary>Respuesta...</details> El atributo open lo muestra expandido por defecto
29 <dfn> Especifica un término que se está definiendo <dfn>Responsive Design</dfn> es... Mejora semántica y accesibilidad
30 <dialog> Representa un cuadro de diálogo o modal <dialog open><p>¡Bienvenido!</p></dialog> Controlado con .showModal() y .close() en JS
31 <div> Contenedor genérico de bloque (sin significado semántico) <div class="card">...</div> Base de casi todo layout con CSS
32 <dl> Lista de descripción (términos + definiciones) <dl><dt>HTML</dt><dd>Lenguaje de marcado</dd></dl> Semántica para pares clave-valor
33 <dt> Define un término en una lista de descripción <dt>Término</dt> Puede haber varios <dt> por cada <dd>
34 <em> Texto enfatizado (con importancia semántica) Esto es <em>muy</em> importante Por defecto cursiva. Preferir sobre <i> cuando hay énfasis
35 <embed> Inserta contenido externo (generalmente plugins) <embed src="mapa.svg" type="image/svg+xml"> Menos usado hoy en día (mejor <object> o <iframe>)
36 <fieldset> Agrupa controles relacionados dentro de un formulario <fieldset><legend>Datos personales</legend>...</fieldset> Mejora accesibilidad con <legend>
37 <figcaption> Leyenda o descripción de un <figure> <figcaption>Figura 1 - Diagrama</figcaption> Debe estar dentro de <figure>
38 <figure> Agrupa contenido autocontenido (imágenes, diagramas, código...) <figure><img src="..."><figcaption>...</figcaption></figure> Elemento semántico HTML5
39 <footer> Contiene pie de página o pie de sección <footer>© 2026 Mi Empresa</footer> Puede estar dentro de <article>, <section>, etc.
40 <form> Define un formulario para enviar datos <form action="/login" method="post">...</form> Atributos clave: action, method, enctype
41 <h1> – <h6> Encabezados de diferentes niveles de importancia <h1>Título principal</h1> <h2>Subtítulo</h2> <h1> es el más importante. Usar jerarquía lógica
42 <head> Contiene metadatos, título, enlaces a CSS/JS, etc. <head><title>Página</title></head> No visible directamente en la página
43 <header> Encabezado introductorio de página o sección <header><h1>Sitio Web</h1><nav>...</nav></header> Semántico HTML5
44 <hr> Línea temática / separador de contenido <hr> Ahora tiene significado semántico (cambio de tema)
45 <html> Elemento raíz de todo el documento HTML <html lang="es">...</html> Atributo lang es muy recomendado
46 <i> Texto en cursiva (sin énfasis semántico) <i>Nombre en latín</i> Usar <em> cuando haya énfasis
47 <iframe> Inserta otra página HTML dentro de la actual <iframe src="https://ejemplo.com" title="Mapa"></iframe> title obligatorio por accesibilidad
48 <img> Inserta una imagen <img src="foto.jpg" alt="Descripción" loading="lazy"> alt es obligatorio. loading="lazy" recomendado
49 <input> Campo de entrada en formulario <input type="email" name="correo" required> Muchos tipos: text, password, checkbox, radio, date, file, etc.
50 <ins> Texto que ha sido insertado <ins datetime="2025-11-15">Nuevo precio</ins> Suele mostrarse subrayado
51 <kbd> Representa entrada de teclado Pulsa <kbd>Ctrl + C</kbd> para copiar Estilo monospace por defecto
52 <label> Etiqueta descriptiva para un control de formulario <label for="nombre">Nombre:</label> <input id="nombre"> Mejora accesibilidad y área de clic
53 <legend> Título de un <fieldset> <legend>Información de contacto</legend> Debe ser el primer hijo de <fieldset>
54 <li> Elemento de una lista <li>Primer ítem</li> Dentro de <ul>, <ol> o <menu>
55 <link> Relaciona el documento con un recurso externo <link rel="stylesheet" href="estilos.css"> Usos comunes: CSS, favicon, canonical, preload...
56 <main> Contenido principal y único de la página <main>...contenido central...</main> Solo debe haber uno visible por página
57 <map> Define un mapa de imagen con áreas clicables <map name="planeta"><area ...></map> Se asocia con <img usemap="#planeta">
58 <mark> Texto resaltado (por relevancia) Resultado de búsqueda: <mark>HTML</mark> Por defecto fondo amarillo
59 <menu> Lista de comandos o acciones <menu><li><button>Guardar</button></li></menu> Soporte limitado en algunos navegadores
60 <meta> Proporciona metadatos sobre la página <meta name="description" content="..."> charset, viewport, author, robots, etc.
61 <meter> Representa una medición escalar (gauge) <meter value="75" min="0" max="100">75%</meter> Atributos: value, min, max, low, high, optimum
62 <nav> Bloque de navegación principal <nav><ul><li><a href="/">Inicio</a></li>...</ul></nav> Semántico HTML5
63 <noscript> Contenido alternativo cuando JavaScript está desactivado <noscript>Por favor activa JavaScript</noscript> Puede ir en <head> o <body>
64 <object> Inserta objeto externo (PDF, SVG, applet...) <object data="documento.pdf" type="application/pdf"></object> Proporciona fallback dentro del elemento
65 <ol> Lista ordenada (numerada) <ol start="5" reversed><li>...</li></ol> Atributos: start, reversed, type
66 <optgroup> Agrupa opciones dentro de un <select> <optgroup label="Europa"><option>España</option>...</optgroup> label es obligatorio
67 <option> Opción dentro de <select> o <datalist> <option value="es">Español</option> Atributos: value, selected, disabled
68 <output> Resultado de un cálculo o acción de formulario <output name="total" for="a b">0</output> Atributos for y form útiles
69 <p> Párrafo de texto <p>Este es un párrafo normal.</p> Elemento de bloque básico
70 <picture> Contenedor para imágenes responsivas con múltiples fuentes <picture><source media="(min-width:800px)" srcset="grande.jpg"><img src="default.jpg"></picture> Ideal para art direction y formatos modernos
71 <pre> Texto preformateado (preserva espacios y saltos) <pre> Texto con formato</pre> Suele usarse con <code>
72 <progress> Barra de progreso <progress value="45" max="100">45%</progress> value y max son los principales atributos
73 <q> Cita corta en línea (agrega comillas automáticas) <q>La vida es bella</q> cite opcional
74 <rp> Paréntesis de fallback para anotaciones ruby <rp>(</rp> Se usa con <ruby> y <rt>
75 <rt> Anotación de pronunciación (principalmente asiático) <rt>kan</rt> Dentro de <ruby>
76 <ruby> Anotaciones ruby (usado en japonés, chino...) <ruby>漢<rt>kan</rt>字<rt>ji</rt></ruby> Combina carácter + pronunciación
77 <s> Texto que ya no es correcto/relevante <s>Precio: $100</s> Ahora $80 Diferente de <del> (semántica distinta)
78 <samp> Muestra salida de un programa o sistema <samp>Error 404: Not Found</samp> Estilo monospace por defecto
79 <script> Inserta código JavaScript <script src="app.js" defer></script> async, defer, type="module" son muy usados
80 <section> Sección temática genérica <section><h2>Capítulo 1</h2>...</section> Debe tener encabezado para semántica
81 <select> Lista desplegable / menú de selección <select name="pais"><option>España</option>...</select> Atributos: multiple, size, required
82 <small> Texto de letra pequeña (fine print, descargos, copyright...) <small>© 2026 Todos los derechos reservados</small> Semántico en HTML5
83 <source> Define fuente multimedia para <audio>, <video> o <picture> <source src="video.webm" type="video/webm"> Se usa con srcset, media, type...
84 <span> Contenedor genérico en línea Precio: <span class="precio">$49</span> Muy usado para aplicar estilos o JS
85 <strong> Texto de fuerte importancia <strong>¡Atención!</strong> Por defecto negrita. Semántico (mejor que <b>)
86 <style> Define estilos CSS dentro del documento <style>body { font-family: Arial; }</style> Normalmente dentro de <head>
87 <sub> Subíndice H<sub>2</sub>O Usado en fórmulas químicas, matemáticas...
88 <summary> Título visible del elemento <details> <summary>¿Cómo funciona?</summary> Obligatorio dentro de <details>
89 <sup> Superíndice x<sup>2</sup> Usado en exponentes matemáticos
90 <table> Define una tabla de datos <table><tr><th>...</th></tr></table> Usar <thead>, <tbody>, <tfoot> cuando corresponda
91 <tbody> Agrupa el cuerpo principal de datos de la tabla <tbody><tr><td>...</td></tr></tbody> Mejora semántica y accesibilidad
92 <td> Celda de datos en una tabla <td colspan="2">Contenido</td> Atributos colspan y rowspan
93 <template> Contenedor de contenido inactivo (para clonar con JS) <template id="item"><div>Plantilla</div></template> No se renderiza hasta que se clona
94 <textarea> Campo de texto multilínea <textarea rows="5" cols="40" maxlength="500"></textarea> placeholder, required, spellcheck...
95 <tfoot> Agrupa el pie de la tabla (totales, resúmenes...) <tfoot><tr><td>Total</td></tr></tfoot> Suele repetirse al imprimir
96 <th> Celda de encabezado en tabla <th scope="col">Nombre</th> scope="row|col|rowgroup|colgroup" mejora accesibilidad
97 <thead> Agrupa los encabezados de la tabla <thead><tr><th>...</th></tr></thead> Se repite al imprimir páginas largas
98 <time> Representa fecha u hora (legible por máquinas) <time datetime="2026-02-25">25 de febrero de 2026</time> datetime es clave para SEO y accesibilidad
99 <title> Define el título de la página (pestaña del navegador) <title>Mi página web</title> Obligatorio dentro de <head>
100 <tr> Define una fila en una tabla <tr><td>Celda 1</td><td>Celda 2</td></tr> Elemento contenedor de celdas (<td> o <th>). Se usa dentro de <table>, <thead>, <tbody> o <tfoot>.
101 <video> Reproductor de video <video controls poster="miniatura.jpg"><source src="vid.mp4" type="video/mp4"></video> poster, controls, autoplay, loop
102 <wbr> Posible salto de línea (word break opportunity) supercalifragilistic<wbr>expialidocious</wbr> Útil en palabras muy largas sin espacios
